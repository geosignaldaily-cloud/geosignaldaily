Privacy Policy

GeoSignal Daily | Last updated: April 19, 2026

1. Overview

GeoSignal Daily ("we", "our", "us") operates an automated geopolitical news channel that publishes AI-generated video content on platforms including TikTok and YouTube. This Privacy Policy explains how we handle data in connection with our TikTok integration.

2. Data We Collect

We do not collect any personal data from TikTok users. Our TikTok integration is used solely to publish video content on behalf of the channel owner (GeoSignal Daily) on a fully automated basis.

The only TikTok data accessed is:

  • Basic account information of the channel owner (to verify account identity during setup)

3. How We Use TikTok API

We use the TikTok Content Posting API exclusively to upload and publish daily news videos to the GeoSignal Daily TikTok account. No user data is collected, stored, shared, or sold.

4. Data Storage

The channel owner's TikTok access token is stored securely on a private server and is used only to authenticate automated video uploads. It is never shared with third parties.

5. Third Parties

We do not share any data with third parties. Our pipeline runs on a private server and does not involve any external data processors beyond the TikTok API itself.
